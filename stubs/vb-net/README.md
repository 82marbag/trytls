## Get started

### Install (linux)
```
..# apt-get install monodevelop mono-vbnc
..# apt-get install mono-vbnc
```
### Run (in linux - similar in other Operating Systems)
```
..# vbnc Run.vb
..# mono Run.exe [host] [port]
```
